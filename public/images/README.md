## Image Asset Structure

Place all site images inside `public/images/` using lowercase kebab-case names.

### Preferred formats

- Use `.webp` for most photos.
- Use `.jpg` or `.jpeg` if the source is already photographic and you do not want to re-export it.
- Use `.png` only when you need transparency or a graphic with sharp edges.

### Naming rules

- Keep names lowercase.
- Use hyphens instead of spaces.
- Use zero-padded sequence numbers for sets: `01`, `02`, `03`.
- Keep one clear primary file per section for the main hero where needed.

### Recommended sizes

- Hero images: at least `2000px` wide.
- Section and card images: `1200px` wide is usually enough.
- Portrait cards: keep a consistent crop ratio before exporting.
